# weatherApp

**weatherApp** is a responsive web application that provides the user with the days weather at a chosen location in addition to that city's five day forecast 

## Features

* HTML
* CSS
* JavaScript
* jQuery
* Bootstrap
* [Open Weather API](http://openweathermap.org/api)
